P1421 小玉买文具

2021-02-01
#include <iostream>
using namespace std;
int main()
{
   int a,b,c,d;
   cin>>a>>b;     //输入
   c=a*10+b;      //a以元为单位，b以角为单位，用c将它们统一单位成角，这样便于处理
   d=c/19;        //用d存放能买到的签字笔的支数
   cout<<d;       //输出
   return 0;
}

这题十分简单，解释都在注释里写过了。唯一需要注意的一点是C++里的除法默认为整除，所以不用担心小数的问题。
